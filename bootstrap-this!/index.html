What Is Bootstrap?

Bootstrap is a CSS framework, open-source, mobile first, and responsive used to build a website. What is great about Bootstrap is that it is free. Bootstrap has their own custom CSS. They have prebuilt components library (eg. buttons, alart, Navbar). Bootstrap is quick and easy to use to build an entire website.

Benefits and Features

Responsive Grid System

Bootstrap has a great grid system. It is very simple to use the grid and makes your website look professional. Using the grid system, you can create a responsive and flexible layout. It is very powerful using the grid system when building mobile first.

Mobile First

One of the great things of using Bootstrap is that it is mobile first. Using their flexbox grid system, you can make your layout mobile first. When creating a website, you want to start with mobile, then tablet, and then desktop. Bootstrap makes it easy to make your website with mobile first.

Open Source

Bootstrap is an open-source. When using Bootstrap, you do not need to deal with purchasing and licensing issuess. Which it is great because it gives you the option to change and experiment with Bootstrap because it is free.

CSS Framework

Bootstrap has their own CSS Framework already build in. This save you time so that you do not have to create your own CSS.

Components

Components are used to make your website better. Bootstrap has lots of components that are build in ready for you to start using them.
