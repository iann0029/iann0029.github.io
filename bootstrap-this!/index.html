What Is Bootstrap?

Bootstrap is a CSS framework, open-source, mobile first, and responsive used to build a website. It is free to use. Bootstrap has their own custom CSS. They have prebuilt components library (eg. buttons, alart, Navbar). Bootstrap is quick and easy to use to build an entire website.

It's Features

Responsive Grid System

Bootstrap has the a great grid system. It is very simple to use the grid and makes your website look like professional. Using the grid system, you can create a responsive and flexible layout. It is very powerful using the grid system when building mobile first.

Mobile First

One of the great things of using Bootstrap is that it is mobile first.
